Replicating Databases

The list of databases supporting replication has grown to include a wide range of databases all with various strengths and locations within an application stack.

Apache CouchDB
The "grandfather" of them all, Apache CouchDB implemented the originating HTTP-based replication spec, is written in Erlang, and has helped spawn this list of various implementions.
Cloudant (BigCouch)
Cloudant offers a CouchDB compatible Database-as-a-Service product built on BigCouch which adds Dynamo-style, multi-node partitioning to the Apache CouchDB core. The BigCouch code-base was donated to the Apache Foundation and is being merged into the core Apache CouchDB project.
Couchbase Lite (TouchDB)
Couchbase Lite (formerly TouchDB) is a lightweight Apache CouchDB-compatible database engine suitable for embedding into mobile or desktop apps. There are both iOS and Android vairants.
PouchDB
PouchDB provides a custom JavaScript API on top of IndexedDB in HTML5 compatible web browsers. While its own, non-HTTP API, it can replicate with many of the other databases that support the replication protocol.
PouchDB Server
PouchDB running via node.js with the same HTTP API as Apache CouchDB.
